This folder contains 5 different datasets for Step 1, each divided into inputs and outputs: 

Input file formatting
---------------------
The first line is: n_u N m beta_v 
The next m lines are y_{v-1}
The next m lines are y_{v}

Output file formatting
----------------------
All m entries of the output are w_{v}. You will compare the results from your implementation
with the data in this file to make sure it's working.
